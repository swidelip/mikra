Usage
----
* Launch all modules
```
mikra.exe all
```
* Launch specific modules
```
mikra.exe browsers
mikra.exe wifi browsers
```
* Get help
```
mikra.exe -h
```
* Quit mode (nothing will be print)
```
mikra.exe all -q
```
* Set hidden attribute to mikra dir
```
mikra.exe all -xh
```
